# SmartCart Clustering System

A simple Machine Learning project that groups similar shopping/customer data using clustering algorithms.


# About Project

This project uses:

* K-Means Clustering
* PCA
* Data Visualization
* Data Preprocessing

The main goal is to find patterns and create groups from data.


# Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook


# Features

* Data Cleaning
* Feature Scaling
* PCA for dimensionality reduction
* Elbow Method and silhouette_score for choosing K value
* K-Means Clustering
* Cluster Visualization
